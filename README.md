# 🤝 Keen Keeper

**Keen Keeper** is a social networking web application where users can connect with friends, discover people, and build meaningful online relationships. It features a modern, responsive UI with smooth navigation and a great user experience.

🌐 **Live Demo:** [https://assingment-07-two.vercel.app/](https://assingment-07-two.vercel.app/)
📦 **Repository:** [https://github.com/md-akash-hasan/Assingment--07](https://github.com/md-akash-hasan/Assingment--07)

---

## ✨ Features

- 👥 Connect and interact with friends
- 🔍 Discover and explore user profiles
- 📊 Data visualization with charts (Recharts)
- 🔔 Toast notifications for real-time feedback
- ⏳ Loading spinners for smooth async experience
- 🧭 Multi-page navigation with React Router
- 📱 Fully responsive design for all screen sizes
- 🎨 Beautiful UI components powered by DaisyUI

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/md-akash-hasan/Assingment--07.git
   cd Assingment--07
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. **Open your browser** and visit `http://localhost:5173`

### Available Scripts

| Command           | Description              |
| ----------------- | ------------------------ |
| `npm run dev`     | Start development server |
| `npm run build`   | Build for production     |
| `npm run preview` | Preview production build |
| `npm run lint`    | Run ESLint checks        |

## 🌍 Deployment

This project is deployed on **Vercel** with a custom `vercel.json` config for SPA routing support.

To deploy your own version:

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com) and import your repository
3. Click **Deploy** — done! ✅
